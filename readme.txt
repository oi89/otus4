Описание:
- Тест открывает личный кабинет otus
- Открывает раздел "О себе"
- В качестве предусловий удаляет все введенные пользовательские данные
- Заново заходит в раздел "О себе"
- Заполняет различные поля страницы: имя, дату рождения, доп. контакты и т.д.
- Очищает куки в браузере
- Еще раз логинится в личном кабинете
- Еще раз открывает раздел "О себе"
- Проверяет, что все введенные данные сохранились

Запуск теста:
Запускать из консоли
mvn clean test -Dbrowser='chrome' -Dlogin='jexoka7653@combcub.com' -Dpassword='qwerty123'
Параметр "-Dbrowser" необязательный. Если он не указан, будет запущен Chrome.
